O MAKPAD é uma ferramenta 100% gratuita que sincroniza textos em tempo real, sem necessidade de logins ou configurações. Ele é uma evolução projetada tanto para a Web quanto para o seu Terminal.

📦 Instalação do CLI "One-Line"

🐧 Linux & macOS

curl -sL https://makpad.mapazero.com/install.sh | bash

🪟 Windows (PowerShell)

iwr -useb https://makpad.mapazero.com/install.ps1 | iex

Como Usar no Terminal:

# Ler todo o conteúdo de um arquivo da nuvem no terminal: $ makpad exemplo # Substituir o conteúdo do arquivo via Pipe ou String: $ echo "Hello MAKPAD" | makpad exemplo $ makpad exemplo "Pausa pro café!" # Anexar ao invés de deletar (Append via Pipe): $ echo "Linha extra" | makpad exemplo --append
